## 重要：body_class filter（必須存在）

必須在 functions.php 裡有這段，否則商品無法渲染：

```php
add_filter('body_class', function($classes) {
    if (is_shop()) {
        $classes = array_diff($classes, ['ast-no-sidebar', 'ast-full-width']);
        $classes[] = 'ast-left-sidebar';
    }
    return $classes;
});
```

## 商品分類結構（重要）

### 正確結構
每個商品只勾**城市**（父分類）和**區**（子分類），不勾⭐外送茶/⭐定點茶。

### 外送茶/定點茶標示
- 存在 ACF `service_type` 欄位（下拉選單）
- 卡片上的「★ 外送茶」從 ACF 讀取，不從分類讀取
- ⭐外送茶、⭐定點茶兩個分類的商品關係已從 DB 移除（2026-06-06）

### ACF service_type 選項（⭐ emoji 前綴）
```
⭐ 外送茶
⭐ 定點茶
⭐ 外送+定點
```
- DB 存的值帶 ⭐ 前綴，程式讀取時統一 str_replace 清除，顯示純文字
- CSS ::before 加 ★，不會雙星

### ACF nationality 欄位（下拉選單）
```
台灣 / 日本 / 韓國 / 越南 / 泰國 / 馬來西亞 / 港澳
俄羅斯 / 大陸 / 新加坡 / 印尼 / 寮國 / 其他國家
```

### 新商品新增流程
1. 右邊分類只勾城市和區
2. 左邊 ACF 欄位：
   - 國籍：下拉選單選擇
   - service_type：下拉選單選 ⭐ 外送茶 / ⭐ 定點茶 / ⭐ 外送+定點

### 商品圖片建議尺寸
- 比例：**2:3（直式）**，例如 860×1290px
- 寬度不低於 430px
- 比例偏離 2:3（如 3:4）時，JS 自動裁切對齊

### 商品分頁設定
- 每頁 12 件：`functions.php` 的 `loop_shop_per_page` + 閱讀設定每頁文章數 = 12
- ⚠️ **設定 → 閱讀 → 每頁文章數** 必須設為 12，否則第一頁只顯示9個導致重複

## 水色雲間商店頁卡片（08-shop-layout.php）

### 卡片顯示順序
```
★ 外送茶  🇹🇼
台中外送茶-米妮
台灣  24歲  167cm  C
```

### 實作方式
- priority 5 hook：輸出 `ts-product-city-row`（ts-product-stype + ts-product-flag）
- priority 15 hook：輸出 `ts-product-tags`
- `wp_head` style 條件：`is_shop() || is_product_category() || is_product()`
- `wp_footer` JS `tsMoveShopMeta` 條件同上
- MutationObserver 監聽 DOM，AJAX 後自動重跑

### 星星實作
```css
.ts-product-stype::before { content: '★ '; color: #FFB800 !important; }
```

### service_type 星號清除（PHP，兩處）
```php
$stype_clean = trim(str_replace(['⭐ ', '⭐', '☆ ', '☆', '★ ', '★'], '', $stype ?: ''));
```

### 篩選器服務類型 chip
```html
<span class="ts-cf-chip" data-val="外送茶">外送茶</span>
<span class="ts-cf-chip" data-val="定點茶">定點茶</span>
<span class="ts-cf-chip" data-val="外送+定點">外送+定點</span>
```

### 篩選器 stype 比對邏輯（外送+定點 納入單選結果）
- 選「外送茶」→ 顯示外送茶 + 外送+定點
- 選「定點茶」→ 顯示定點茶 + 外送+定點
- 選「外送+定點」→ 只顯示外送+定點

### 圖片比例自動裁切（JS）
- 比例 > 0.72（寬於 2:3，如 3:4）→ 套用 aspect-ratio:2/3, object-fit:cover
- 比例 ≤ 0.72 → 不處理

### 全站 emoji 尺寸限制（tianshang-style.css 最底部）
```css
img.emoji { width:16px !important; height:16px !important; max-width:16px !important; max-height:16px !important; }
```

### 篩選器按鈕
- 文字：`⚡ 精準配對`，霓虹效果 `@keyframes tsShopNeon`
- ⚠️ `@keyframes` 必須在 `@media` query **外面**

### 縮圖尺寸
- 430×0（等比例）
- DB：`woocommerce_thumbnail_image_width = 430`，`woocommerce_thumbnail_cropping = uncropped`
